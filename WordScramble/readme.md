Word Scramble

This game gives user a word then expects user to come up with a new word made up with the given word, but there are rules: 
1. Users CANNOT come up with words that are less than 3 letters, and the same as the given word,
2. The word that the user types MUST be a real word and can be written by the given word. 
Other than that, user gets a point for each word they typed.
When the user presses the "start" button, the word changes and the word thst user typed appears with the number of letters next to the word.
